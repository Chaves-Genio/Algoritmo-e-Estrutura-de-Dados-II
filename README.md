# Algoritmo-e-Estrutura-de-Dados-II
Estrutura de Dados II - Pilha e Fila

Autor: António Chaves N. Monteiro 

Este repositório contém a implementação das estruturas de dados Pilha e Fila, conceitos abordados na aula 1.

O objectivo principal é demonstrar o funcionamento dessas estruturas respeitado o conceito de TAD.

PILHA
Estrutura de dados linear, baseada no princípio LIFO(Last In, First Out).
Operações Implementadas:
push - inserção do elemento na pilha
pop - Remoção do elemento 
peek - Consultar elemento no Topo
isEmpty - Verificar se está vazia

FILA
Estrutura de dados linear, baseada no princípio FIFO(First In, First Out).
Operações Implementadas:
enqueue - inserção do elemento na pilha
dequeue - Remoção do elemento 
front - Consultar primeiro elemento
isEmpty - Verificar se está vazia

Os programas foram criados na linguagem de programação java
